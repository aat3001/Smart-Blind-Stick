# Smart Blind Stick
<br>
A prototype assistive device designed to help visually impaired individuals detect nearby obstacles using an ultrasonic sensor and buzzer alert system.
<br>
## Features<br>
- Arduino-based obstacle detection<br>
- Ultrasonic distance sensing<br>
- Buzzer for real-time alerts<br>
- Lightweight PVC body design<br>
<br>
## Working<br>
The ultrasonic sensor continuously measures distance.  <br>
If an obstacle is detected within 15 cm, the buzzer activates to alert the user.<br>
